# Console OS Vue Portal

## Build

add the ```vue-cli-plugin-os``` to the dev dependencies.

```bash
> npm install vue-cli-plugin-console-os -D
# or
> yarn add vue-cli-plugin-console-os -D
```

## Config

add the Config in vue.config.js

```
module.export = {
  pluginOptions: {
    consoleOs: {
      id: 'xxxx-xxxxxx'
    }
  }
}
```
